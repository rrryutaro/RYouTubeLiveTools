最終更新日: 2026年7月4日

言語: [English](terms-of-service.md) | 日本語

## 対象範囲

本利用規約は、ライブ配信支援を目的とした Windows デスクトップツール群である RYouTubeLiveTools に適用されます。

このリポジトリで YouTube API Services を利用するツールは RCommentHub です。RCommentHub は、YouTube Live のチャットメッセージを取得、表示、フィルタリング、ログ保存、読み上げします。このリポジトリに含まれる他のツールは、YouTube API Services を直接利用しません。RRoulette は、ユーザーが機能を有効にした場合に限り、RCommentHub からローカル連携経由でコメント関連データを受け取ることがあります。

## 無償提供

RYouTubeLiveTools は、現時点では無償で提供されています。現時点で、有料サブスクリプション、アプリ内購入、広告収益機能はありません。

本ソフトウェアは MIT License に基づいて配布されます。本利用規約は、MIT License によって許諾される権利と矛盾することを意図していません。

RCommentHub の一部機能は、YouTube Data API v3 のライブチャット関連 API を含む YouTube API Services を利用します。これらの機能を使用する場合、ユーザーは次の規約およびポリシーにも従う必要があります。

- [YouTube Terms of Service](https://www.youtube.com/t/terms)

ユーザーは、自身の Google アカウント、Google Cloud プロジェクト、OAuth クライアント設定、API キー、YouTube Live 配信データを、Google および YouTube の規約に従って使用する責任があります。
