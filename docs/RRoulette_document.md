# RRoulette ドキュメント

作成日: 2026/03/30
バージョン: 0.2.0

YouTube ライブ配信向けのルーレットアプリです。

---

## 起動方法

| 方法 | 手順 |
|------|------|
| Python 直実行 | `python roulette.py`（RRoulette フォルダ内） |
| EXE | `build_exe.bat` でビルド後、`dist\RRoulette.exe` を実行 |
| ランチャー | `launcher.py` の「▶ Python」または「▶ EXE」ボタン |

---

## 画面構成

```
┌──────────────────────────────────────────────────────┐
│  [ルーレット]          ‖  [項目リスト]  ‖  [設定]    │
│  ┌────────────────┐  ‖  [グループ ▼] ‖  スピン設定  │
│  │                │  ‖  項目A        ‖  音量        │
│  │   ルーレット   │  ‖  項目B        ‖  音パターン  │
│  │    ホイール    │  ‖  ...          ‖  文字表示    │
│  │                │  ‖  [編集][保存] ‖  マーカー    │
│  └────────────────┘  ‖               ‖              │
│    （結果オーバーレイ）                               │
└──────────────────────────────────────────────────────┘
```

- 項目リスト・設定パネルは右クリックメニューから表示/非表示を切替可能
- 中央のバー（‖）をドラッグして各パネルの幅を調整できる

---

## ルーレット操作

| 操作 | 方法 |
|------|------|
| **スピン開始** | ホイールをクリック または スペースキー |
| **早期停止（2倍速）** | スピン中に素早く2回クリック/スペース |
| **即時停止** | スピン中に素早く3回クリック/スペース |
| **マーカー移動** | マーカー（金色の三角）をドラッグ |

結果はホイール中央にオーバーレイ表示されます。次のスピン開始時に消えます。

## 項目リスト操作

### グループ（項目セット）の管理

コンボボックス（項目リスト上部）でグループを管理します。ポップアップウィンドウは一切表示されません。

| 操作 | 方法 |
|------|------|
| **グループ切替** | コンボボックスをクリックして一覧から選ぶ |
| **グループ追加** | 一覧の末尾「＋ 新規グループを追加...」を選択 → 自動作成＆名前入力状態 → Enter |
| **グループ名変更** | コンボボックスのテキストを直接書き換えて Enter |
| **グループ削除** | コンボボックスのテキストを全消しして Enter（グループが1つの場合は削除不可） |
| **編集キャンセル** | Escape で元の名前に戻る |

グループの内容・現在のグループ名はアプリ終了時に自動保存されます。

### 項目の ON/OFF

| 操作 | 方法 |
|------|------|
| **ON/OFF 切り替え** | 各項目左のチェックボックスをクリック |

OFF にした項目はルーレットから除外されます。グループ内の全項目が OFF の場合はスピンできません。

### 確率設定（項目ごと）

各項目行の「確率」欄で確率設定方式を選択できます。

| 方式 | 説明 |
|------|------|
| デフォルト | 全 ON 項目を等確率で扱う |
| 固定% | この項目の最終確率を直接指定する（例: 30 と入力 → 30%）。Enter または入力欄からフォーカスを外すと反映される |
| 重み | この項目の重みを変更し、全体を正規化して最終確率を決める（例: 重み 2 → 他の項目の 2 倍の確率） |

**固定% の注意点:**
- 0% と 100% は指定できません（0% は ON/OFF で表現します）
- 複数項目に固定確率を設定する場合、合計が 100% 以上になる入力は拒否されます

**重みの選択肢:**
- 1 未満: 0.25 / 0.5 / 0.75
- 1（デフォルト）
- 1 より大きい: 1.5 / 2.0 / 2.5 / … （ON 項目数まで 0.5 刻み）

### 分割設定（項目ごと）

各項目行の「分割」欄でルーレット上の区画数を設定できます。

| 設定 | 説明 |
|------|------|
| 1（デフォルト） | その項目は 1 つの区画として表示される |
| 2〜10 | 同じ確率の区画に分割して表示する（ルーレット上の見た目が変わる） |

分割しても最終確率は変わりません。
分割した項目は標準配置で等間隔に配置されます。

### ルーレット項目の編集

| 操作 | 方法 |
|------|------|
| **編集モードへ** | 「編集」ボタンをクリック → テキスト入力欄に切り替わる（1行1項目） |
| **項目を追加・変更** | テキスト欄を直接編集（Ctrl+Z で Undo、Ctrl+Y で Redo） |
| **保存** | 「保存」ボタンをクリック → 空白行は自動削除、リストボックス表示に戻る |

## ウィンドウ操作

| 操作 | 方法 |
|------|------|
| **移動** | ルーレット・背景エリアをドラッグ |
| **リサイズ** | 右下グリップをドラッグ |
| **サイズプロファイル切替** | 右クリックメニュー → S / M / L |
| **常に最前面** | 右クリックメニュー → 常に最前面 |
| **項目リスト 表示/非表示** | 右クリックメニュー → 項目リストを表示/非表示 |
| **設定パネル 表示/非表示** | 右クリックメニュー → 設定を表示/非表示 |
| **最小化** | 右クリックメニュー → 最小化 |
| **終了** | 右クリックメニュー → 終了（設定・グループデータを自動保存） |

## 設定パネル

右クリックメニュー → 「設定を表示」で開きます。縦にスクロール可能。

| 設定項目 | 内容 |
|----------|------|
| **操作設定** | スピン時間・ダブル停止時間・トリプル停止時間（各スライダー） |
| **音量** | 0〜100 % |
| **スピン音** | クリック / ドラム / コイン / ソフト / 消音（試聴ボタンあり） |
| **決定音** | ファンファーレ / ベル / カジノ / チャイム / ビクトリー（試聴ボタンあり） |
| **ルーレット文字表示** | 表示方向・文字サイズの扱いをコンボボックスで選択 |
| **マーカー位置** | 上/右/下/左/任意（ドラッグでも変更可） |
| **配置設定** | 標準配置に戻す / 今すぐランダム配置 / spinごとにランダム配置 |

## 保存データ（roulette_settings.json）

EXE と同じフォルダ（`dist\`）に自動生成されます。以下の情報が保存されます。

- ウィンドウ位置・サイズ
- サイズプロファイル
- 常に最前面 ON/OFF
- 各パネルの表示状態・幅
- スピン設定・音量・音パターン
- 文字表示設定
- マーカー位置
- **グループ一覧と各グループの項目**（ON/OFF・確率設定・分割設定を含む）
- **現在選択中のグループ名**
- **spinごとにランダム配置 ON/OFF**

各グループの項目は「テキスト / ON/OFF / 確率設定 / 分割数」のセットで保存されます。設定ファイルのバージョンが古い場合（テキストのみ形式）は自動的に移行されます。

## 既知の問題

### サッシュ（仕切りバー）がウィンドウ外に押し出される

**再現手順:**

1. ウィンドウを大きい状態（M または L サイズ）にする
2. 項目リストと本体の境界バーをドラッグして項目リストを大きく広げる
3. ウィンドウを小さくリサイズする

**症状:** 項目リストの幅がウィンドウ幅を超え、境界バーが画面外に押し出されて操作できなくなる。

**回避策:**

- 右クリックメニュー → 「項目リストを非表示」→ 「項目リストを表示」で表示を切り替えると、サッシュが画面内に戻る場合があります
- それでも戻らない場合は、ウィンドウを再度広げてからバーをドラッグして幅を調整してください
- または右クリックメニュー → サイズ S/M/L で大きいプロファイルに切り替えてから調整してください

## ビルド（EXE 作成）

ランチャーの「🔨 Build」ボタン、または `RRoulette\build_exe.bat` を直接実行してください。
ビルド完了後、`RRoulette\dist\RRoulette.exe` が生成されます。

依存ライブラリ: `pygame`, `numpy`, `pyinstaller`
