# RCommentHub

YouTube Live のコメントを受信・一覧表示・詳細確認するハブツール。
将来的には RRoulette 等の他ツールへのコメント連携基盤となる。

## バージョン

v0.1.0

## 機能概要

- YouTube Live のコメントをリアルタイム受信（liveChatMessages.streamList）
- コメント一覧表示（種別・投稿者・本文・権限フラグ等）
- 選択コメントの詳細確認（Raw JSON 含む）
- 種別・属性・文字列・状態によるフィルタ
- 接続状態・配信情報の表示
- 内部ログ表示

## 実行方法

```bash
cd RCommentHub
python rcommenthub.py
```

## ファイル構成

| ファイル | 説明 |
|---|---|
| `rcommenthub.py` | メインウィンドウ・アプリケーションロジック |
| `constants.py` | 定数定義（色・ラベル・カラム定義等） |
| `rcommenthub_settings.json` | ウィンドウ位置・ペイン配置の保存（自動生成） |

## 画面構成

```
┌─────────────────────────────────────────────────────────┐
│ ヘッダー: 接続状態 / 配信タイトル / 動画ID / 統計        │
├──────────┬──────────────────────────────┬───────────────┤
│ フィルタ │ コメント一覧（Treeview）      │ 詳細ペイン    │
│          │                              │ A. 基本情報   │
│ 種別     │ No / 受信時刻 / 投稿時刻 /   │ B. 投稿者情報 │
│ 投稿者   │ 種別 / 投稿者名 / 本文 /     │ C. 種別別詳細 │
│ 文字列   │ ChID / 権限 / MsgID / 状態   │ D. 内部処理   │
│ 状態     │                              │ E. Raw JSON   │
├──────────┴──────────────────────────────┴───────────────┤
│ ログエリア                                               │
└─────────────────────────────────────────────────────────┘
```

## 依存ライブラリ

- Python 3.11+
- tkinter（標準ライブラリ）

YouTube API 接続時は追加で google-api-python-client 等が必要（将来実装）。
