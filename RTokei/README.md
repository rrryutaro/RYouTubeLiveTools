# RTokei

バージョン: v0.1.0

デスクトップに常駐するデジタル時計ツールです。
最前面表示・背景透過・フォントや色のカスタマイズに対応し、配信画面への重ね表示などに活用できます。

---

## ダウンロードして使う（一般ユーザー向け）

Python などの開発環境は不要です。

1. [Releases](../../releases) ページから `RTokei.exe` をダウンロード
2. ダウンロードした `RTokei.exe` をダブルクリックで起動

> Windows によって「このアプリの実行についての警告」が表示された場合は、
> 「詳細情報」→「実行」をクリックしてください。

---

## 使い方

### 基本操作

| 操作 | 方法 |
|------|------|
| 設定を開く | 右クリック → 設定を開く |
| サイズ変更 | 右クリック → サイズ（極小/小/中/大/極大） |
| 日付表示切替 | 右クリック → 日付を表示 |
| 時刻表示切替 | 右クリック → 時刻を表示 |
| 最前面切替 | 右クリック → 常に最前面 |
| 背景透過切替 | 右クリック → 背景透過 |
| タイトルバー切替 | 右クリック → タイトルバー |
| ウィンドウ移動 | ドラッグ（タイトルバーなし時も可） |
| リサイズ | 右下のグリップをドラッグ |
| 最小化 | 右クリック → 最小化 |
| 終了 | 右クリック → 終了 |

### 設定画面でできること

#### 表示設定

| 項目 | 内容 |
|------|------|
| 表示項目 | 日付・時刻それぞれの表示/非表示を切替。どちらか一方は必ず表示されます |
| 時刻の形式 | 下表の5種類から選択 |

**時刻の表示形式**

| 選択肢 | 表示例 |
|--------|--------|
| HH:MM:SS | 13:45:30 |
| HH:MM | 13:45 |
| HH時MM分SS秒 | 13時45分30秒 |
| HH時MM分 | 13時45分 |
| MM:SS | 45:30 |

#### フォント

| 項目 | 内容 |
|------|------|
| フォント名 | ドロップダウンからシステムフォントを選択（直接入力も可） |
| フォントサイズ | 日付・時刻それぞれ個別に指定（8〜200） |
| スタイル | 太字・斜体の ON/OFF |

#### カラー

| 項目 | 内容 |
|------|------|
| 日付の色 | 日付テキストの文字色（カラーピッカー） |
| 時刻の色 | 時刻テキストの文字色（カラーピッカー） |
| 背景色 | ウィンドウ背景色（カラーピッカー） |

#### サイズプリセット

極小 / 小 / 中 / 大 / 極大 の5段階。選択するとウィンドウサイズとフォントサイズが一括で変わります。

#### ウィンドウ

| 項目 | 内容 |
|------|------|
| 常に最前面 | 他のウィンドウより前面に固定 |
| 背景透過 | 背景色と同色の領域を透明にする |
| タイトルバー | タイトルバーの表示/非表示 |

### 注意

- 「背景透過」は背景色と同色の部分が透明になります
- タイトルバーを非表示にした場合はドラッグで移動してください
- 位置・サイズは終了時に自動保存されます
- 日付・時刻の表示切替はメニューからも設定画面からも行えます

### OBSで背景透過を使う場合

OBSのウィンドウキャプチャに背景透過を反映させるには、キャプチャ方法の変更が必要です。

1. OBSのソース一覧でウィンドウキャプチャソースを右クリック →「プロパティ」を開く
2. **「キャプチャ方法」を「Windows 10 (1903以降)」に変更**する
3. 「OK」で閉じる

> 「BitBlt」など他のキャプチャ方法では背景透過がOBSに反映されません。
>
> **動作確認バージョン**: OBS Studio 32.1.0 (64 bit)
> 他のバージョンでは設定項目の名称・位置が異なる場合があります。

## トラブルシューティング

**ウィンドウが画面外に出てしまった**
→ 設定ファイル（`rtokei_settings.json`）を削除して起動すると初期位置にリセットされます。

**OBSに映らない**
→ OBSの「ウィンドウキャプチャ」で `RTokei` を選択してください。「ゲームキャプチャ」では映りません。

**背景透過がOBSに反映されない**
→ OBSのウィンドウキャプチャソースのプロパティを開き、**「キャプチャ方法」を「Windows 10 (1903以降)」に変更**してください。
「BitBlt」など他のキャプチャ方法では背景透過がOBSに反映されません。
この手順は OBS Studio 32.1.0 (64 bit) で動作確認済みです。他のバージョンでは項目名・位置が異なる場合があります。

**フォント設定が反映されない**
→ 設定画面でフォントを変更後、「OK」で設定画面を閉じてください。変更はリアルタイムでプレビューされます。

## ソースから実行 / ビルドする（開発者向け）

### 必要環境
- Python 3.10 以上
- Windows 10 / 11

### 依存ライブラリのインストール

```
pip install pyinstaller
```

> tkinter は Python 標準ライブラリに含まれています。
> 「tkinter が見つからない」と出る場合は Python を再インストールし、
> インストール時に「tcl/tk and IDLE」にチェックを入れてください。

### EXE のビルド

`build_exe.bat` をダブルクリックするだけでビルドできます。

または手動でビルドする場合:

```
pyinstaller RTokei.spec
```

ビルドが完了すると `dist\RTokei.exe` が生成されます。

### アイコンを付けたい場合

`.ico` ファイルを用意して `RTokei.spec` の以下の行をコメントアウト解除してください:

```
# icon='rtokei.ico',
```

### 設定ファイルについて

`rtokei_settings.json` は EXE と同じフォルダに自動生成されます。
削除するとデフォルト設定にリセットされます。
