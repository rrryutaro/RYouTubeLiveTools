# RSheetsViewer

バージョン: v0.1.0

配信中にブラウザの余計な情報を映さず、Google スプレッドシートだけを操作するための専用GUIツールです。

---

## ダウンロードして使う（一般ユーザー向け）

Python などの開発環境は不要です。

1. [Releases](../../releases) ページから `RSheetsViewer.exe` をダウンロード
2. ダウンロードした `RSheetsViewer.exe` をダブルクリックで起動

> Windows によって「このアプリの実行についての警告」が表示された場合は、
> 「詳細情報」→「実行」をクリックしてください。

---

## 使い方

### 初回起動
- URLの入力ダイアログが表示されます
- Google スプレッドシートのURLを貼り付けて「OK」をクリック
- Googleアカウントのログインが必要な場合は自動的にログイン画面が開きます

### 2回目以降
- 前回開いたスプレッドシートが自動的に開きます
- ウィンドウのサイズ・位置も前回の状態が復元されます

---

## 操作方法

| 操作 | 方法 |
|------|------|
| URLを変更 | ツールバーの「🔗 URLを変更」ボタン または `Ctrl+L` |
| 再読み込み | ツールバーの「↻」ボタン または `F5` |
| 戻る | ツールバーの「◀」ボタン または `Alt+←` |
| 進む | ツールバーの「▶」ボタン または `Alt+→` |
| 全画面表示 | `F11` |
| 常に手前に表示 | ツールバーの「📌」ボタン（📌=固定中 / 📍=未固定）。設定は保存されます |

---

## セキュリティについて

- **専用プロファイル**: ブラウザデータ（Cookie・ログイン情報）は通常のChromeとは完全に分離された専用フォルダに保存されます
- **データ保存場所**: `%APPDATA%\SheetsViewer\`
  - `sheetsviewer_settings.json` … 最後に開いたURL・ウィンドウ位置のみ保存（パスワード等は含みません）
  - `chrome_profile\` … Cookieなどのブラウザデータ（Googleログインセッション）

## トラブルシューティング

### 起動しない場合
- Google Chrome がインストールされているか確認してください
- インストールされていない場合は https://www.google.com/chrome/ からインストールしてください

### ログインが維持されない場合
- `%APPDATA%\SheetsViewer\chrome_profile` フォルダが存在するか確認してください
- アンチウイルスソフトがフォルダへの書き込みをブロックしている場合があります

### 画面が真っ白になる場合
- F5キーで再読み込みしてください
- それでも解決しない場合はURLを再入力してください（Ctrl+L）

## Chrome の標準機能で代替する方法

このツールと同等のことは、Google Chrome の標準機能だけでも実現できます。

### 手順

1. Google スプレッドシートを Chrome で開く
2. 右上の「︙」メニュー →「キャスト、保存、共有」→「ページをアプリとしてインストール」
3. 任意の名前を付けて「インストール」をクリック

インストール後はデスクトップのショートカットまたはスタートメニューから起動でき、
タブバー・アドレスバーなしの独立したウィンドウでスプレッドシートが開きます。
OBS でのウィンドウキャプチャなど、ブラウザのUIを映したくない用途には、
この方法で十分に対応できます。

> このツール（RSheetsViewer）はそもそも上記の Chrome 標準機能を知らずに作成したものです。
> **同等の機能が Chrome に最初から備わっている**ため、このツールを特別に使用する必要はありません。
> Chrome の標準機能で対応することをおすすめします。

## ソースから実行する（開発者向け）

### 必要環境
- Windows 10 / 11
- Python 3.10 以上

### 1. Python のインストール（未インストールの場合）

https://www.python.org/downloads/ から最新版をダウンロードし、
**「Add Python to PATH」にチェックを入れて**インストールしてください。

### 2. 依存パッケージのインストール

```
pip install -r requirements.txt
```

### 3. 起動

```
python sheets_viewer.py
```

### EXE のビルド

```
pyinstaller RSheetsViewer.spec
```

ビルドが完了すると `dist\RSheetsViewer.exe` が生成されます。

### トラブルシューティング（開発者向け）

起動しない場合:
```
pip install --upgrade PyQt6 PyQt6-WebEngine
```
