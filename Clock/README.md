# 時計アプリ — ビルド手順

## 必要環境
- Python 3.9 以上（3.11 推奨）
- Windows 10 / 11

---

## 1. 依存ライブラリのインストール

```
pip install pyinstaller
```

tkinter は Python 標準ライブラリに含まれています。
※ もし「tkinter が見つからない」と出る場合は Python を再インストールし、
   インストール時に「tcl/tk and IDLE」にチェックを入れてください。

---

## 2. EXE ビルド（ワンライナー）

clock.py と同じフォルダで以下を実行：

```
pyinstaller --onefile --noconsole --name "時計アプリ" clock.py
```

または spec ファイルを使う場合：

```
pyinstaller clock.spec
```

ビルドが完了すると `dist\時計アプリ.exe` が生成されます。

---

## 3. アイコンを付けたい場合

.ico ファイルを用意して以下のオプションを追加：

```
pyinstaller --onefile --noconsole --icon=clock.ico --name "時計アプリ" clock.py
```

---

## 4. 設定ファイルについて

`clock_settings.json` は EXE と同じフォルダに自動生成されます。
削除するとデフォルト設定にリセットされます。

---

## 機能一覧

| 操作 | 方法 |
|------|------|
| 設定を開く | 右クリック → 設定 |
| サイズ変更 | 右クリック → サイズ（極小/小/中/大/極大） |
| 最前面切替 | 右クリック → 常に最前面 |
| 背景透過切替 | 右クリック → 背景透過 |
| タイトルバー切替 | 右クリック → タイトルバー |
| ウィンドウ移動 | ドラッグ（タイトルバーなし時も可） |
| 終了 | 右クリック → 終了 |

## 設定画面でできること

- 表示形式の変更（HH:MM:SS / HH:MM / 日付+時刻 / 曜日表示など）
- フォント名・サイズ・太字・斜体の変更
- 文字色・背景色の変更（カラーピッカー付き）
- サイズプリセット選択（極小〜極大）
- 最前面・透過背景・タイトルバーの ON/OFF

## 注意

- 「背景透過」は背景色と同色の部分が透明になります
- タイトルバーを非表示にした場合はドラッグで移動してください
- 位置・サイズは終了時に自動保存されます
