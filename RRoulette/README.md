# RRoulette

配信や動画収録で「こんなのあったらいいな」と思ったものを実現しようとして作成しているルーレットアプリです。  

---

## 主な特徴

- シンプルなルーレットとしてすぐ使える
- 項目ごとの有効 / 無効切り替えができる
- 確率変更（固定確率 / 重み付け）ができる
- 項目の分割表示ができる
- 配置のランダム化に対応
- ログ履歴を表示できる
- 表示方法や見た目を調整できる
- 効果音や停止演出を調整できる

---

## できること

### 1. 基本のルーレット機能

- 項目を登録してルーレットを回す
- 結果をログとして残す
- 連続して何度でも抽選できる

通常のルーレットとしてそのまま使えます。

---

### 2. 項目ごとの有効 / 無効切り替え

項目ごとに ON / OFF を切り替えられます。

例えば、

- 今回は使わない候補を一時的に除外したい
- 一部の候補だけを対象にしたい

といった使い方ができます。

項目を削除しなくても、その場だけ候補から外せるようになっています。

---

### 3. 確率変更

項目ごとに抽選確率の調整ができます。

#### 固定確率
特定の項目に対して、直接パーセンテージを指定する方式です。

例:
- A を 50%
- B を 20%
- C を残りの比率で自動配分

#### 重み付け
「当たりやすさ」を相対的に調整する方式です。

例:
- A を通常の2倍当たりやすくする
- B を通常の半分にする

「しっかり確率を調整したい」場合にも、  
「なんとなく当たりやすくしたい」場合にも使えるようにしています。

---

### 4. 項目の分割

1つの項目を複数のセグメントとして分割表示できます。

例:
- `A` を 3 分割
- `B` を 2 分割

これにより、

- 見た目として分散して配置したい
- 同じ項目を複数箇所に見せたい
- 演出的に散らしたい

といった使い方ができます。

※ 最終結果は、停止時点でポインターが指しているセグメントの元項目で判定されます。

---

### 5. 配置のランダム化

ルーレット内の並び順をランダムに変更できます。

#### 1回だけランダム
現在の並びをその場でランダム化します。

#### 常時ランダム
スピン開始のたびに、配置をランダムに変更します。

「毎回見た目を変えたい」  
「同じ並びで固定される印象を減らしたい」  
といった用途向けです。

---

### 6. ログ履歴表示

抽選結果はログとして履歴に残せます。

- 何が選ばれたかを確認できる
- 配信中に過去結果を見せられる
- ログの表示位置や見え方も調整可能

ログ表示は、ルーレットとの重なり方や表示方法もある程度調整できます。

---

### 7. 表示方法の調整

文字の表示やルーレットの見え方を変更できます。

主な調整例:

- テキストの表示モード変更
- 文字方向の変更
- ドーナツ穴の有無
- 背景透過
- ポインター位置の調整
- 各種表示サイズの変更

まだ調整中の部分もあるため、今後も少しずつ改修していく予定です。

---

### 8. 効果音・演出

スピン中や決定時の効果音を変更できます。

- 内蔵効果音の切り替え
- 消音
- カスタム音声の利用（対応環境時）

見た目だけでなく、演出面も調整できるようにしています。

---

## 使い方

### Python で起動する場合

```bash
python rroulette.py
```

### 実行ファイル（exe）を使う場合

配布されている exe を起動してください。

---

## 基本的な使い方

1. 項目を入力する  
2. 必要に応じて ON / OFF や確率を調整する  
3. 必要なら見た目や配置を調整する  
4. スピンする  
5. 結果を確認する  

---

## 設定・保存

本アプリでは、各種設定やデータを保存できます。

- アプリ設定の保存 / 復元
- 項目リストの保存 / 読み込み
- JSON インポート / エクスポート
- ログの出力

---

## 注意事項

- 一部機能は実行環境によって挙動が変わる場合があります
- 開発途中のため、今後設定項目や挙動が変更される可能性があります
- バージョン差異により、古い設定ファイルがそのまま使えない場合があります
