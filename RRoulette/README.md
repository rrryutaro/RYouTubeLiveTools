# RRoulette

バージョン: v0.1.0

YouTubeライブ配信向けルーレットツール。OBSのウィンドウキャプチャに対応しています。

---

## ダウンロードして使う（一般ユーザー向け）

[Releases](../../releases) ページから最新の `RRoulette.exe` をダウンロードして実行してください。インストール不要です。

---

## 使い方 / 操作方法

### 基本操作

| 操作 | 動作 |
|------|------|
| 画面クリック または スペースキー | スピン開始 / 停止 |
| スピン中に2連打（400ms以内） | ダブル停止（設定秒数で減速） |
| スピン中に3連打 | トリプル停止（即時停止も可） |
| 右クリック | メニューを表示 |

### 項目リスト

- 右側のサイドバーで項目を管理します
- **編集ボタン** → テキスト欄で項目を編集（1行1項目） → **保存ボタン**
- コンボボックスでグループを切り替え・追加・削除できます

### マーカー

- ゴールドの三角マーカーをドラッグして任意の位置に移動できます
- 設定パネルのプリセット（上/右/下/左）でも変更できます

### ログ表示

- スピン結果が画面左端にリアルタイムで表示されます（新しいものが上）
- 結果にカーソルを合わせると、そのときのグループ・項目構成がポップアップ表示されます
- 設定パネルの「ルーレット上にログを表示する」でON/OFFを切り替えられます

### 設定パネル

右クリックメニューから「設定パネル」を開くと以下の設定ができます:

- スピン時間（通常 / ダブル停止 / トリプル停止）
- 音量・スピン音・決定音のパターン
- ルーレット文字の表示方向・サイズ
- マーカー位置プリセット
- ログ設定（日時記録・オーバーレイ表示）
- 項目リストのインポート / エクスポート

### ログ出力

右クリックメニューから結果ログをファイルに保存できます:

- **結果のみ（.txt）**: シンプルなテキスト形式
- **グループ・項目付き（.json）**: グループ名と項目構成を含む詳細形式

## トラブルシューティング

**ウィンドウが画面外に出てしまった**
→ 設定ファイル（`roulette_settings.json`）を削除して起動すると初期設定にリセットされます。

**音が出ない**
→ 設定パネルの音量スライダーを確認してください。スピン音が「消音」になっている可能性があります。

**OBSに映らない**
→ OBSの「ウィンドウキャプチャ」で `RRoulette` を選択してください。「ゲームキャプチャ」では映りません。

**背景透過がOBSに反映されない**
→ OBSのウィンドウキャプチャソースのプロパティを開き、**「キャプチャ方法」を「Windows 10 (1903以降)」に変更**してください。
「BitBlt」など他のキャプチャ方法では背景透過がOBSに反映されません。
この手順は OBS Studio 32.1.0 (64 bit) で動作確認済みです。他のバージョンでは項目名・位置が異なる場合があります。

以下の機能は将来のアップデートで対応予定です。

| 課題 | 概要 |
|------|------|
| 右クリックメニューのOBS対応 | 現状は別ウィンドウのためOBSに映らない。視聴者が配信者の操作を確認できるよう、メニュー内容をキャンバス上に描画する方式への変更を検討 |

## ソースから実行する（開発者向け）

### 必要環境

- Windows 10 / 11
- Python 3.10 以上
- 依存パッケージ: `pygame`, `numpy`

### 依存パッケージのインストール

```
pip install -r requirements.txt
```

### 起動

```
python rroulette.py
```

### EXEのビルド

`build_exe.bat` をダブルクリックするだけでビルドできます。

または手動でビルドする場合:

```
pyinstaller RRoulette.spec
```

ビルドが完了すると `dist\RRoulette.exe` が生成されます。

### 設定ファイルについて

`roulette_settings.json` は EXE と同じフォルダに自動生成されます。
削除すると初期設定にリセットされます。
