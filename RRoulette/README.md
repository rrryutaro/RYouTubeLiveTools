# RRoulette

配信や動画収録で「こんなのあったらいいな」と思ったものを実現しようとして作成しているルーレットアプリです。

---

## 主な特徴

- シンプルなルーレットとしてすぐ使える
- 項目ごとの有効 / 無効切り替えができる
- 確率変更（固定確率 / 重み付け）ができる
- 項目の分割表示ができる
- 配置のランダム化に対応
- ログ履歴を表示できる
- 表示方法や見た目を調整できる
- 効果音や停止演出を調整できる（音量個別調整対応）
- 被りなし連続抽選ができる（指定回数・重複なし）
- デザインエディタで色・フォントを編集できる
- 複数ルーレットを同時に管理できる
- 画面範囲を選択してOCRし、ゲーム内の選択肢などを項目テキストとしてコピーできる
- チケット管理ができる（効果付きチケットの発行・使用・履歴）
- スピンのリプレイを保存・再生できる
- 外部ツールと連携してスピンやチケット追加を自動実行できる

---

## 起動方法

配布されている exe を起動してください。

---

## 基本的な操作の流れ

1. 項目を入力する
2. 必要に応じて ON / OFF や確率を調整する
3. 必要なら見た目や配置を調整する
4. スピンする
5. 結果を確認する

---

## 機能詳細

### 1. 基本のルーレット機能

- 項目を登録してルーレットを回す
- 結果をログとして残す
- 連続して何度でも抽選できる

通常のルーレットとしてそのまま使えます。

---

### 2. 項目ごとの有効 / 無効切り替え

項目ごとに ON / OFF を切り替えられます。

例えば、

- 今回は使わない候補を一時的に除外したい
- 一部の候補だけを対象にしたい

といった使い方ができます。

項目を削除しなくても、その場だけ候補から外せるようになっています。

---

### 3. 確率変更

項目ごとに抽選確率の調整ができます。

#### 固定確率
特定の項目に対して、直接パーセンテージを指定する方式です。

例:
- A を 50%
- B を 20%
- C を残りの比率で自動配分

#### 重み付け
「当たりやすさ」を相対的に調整する方式です。

例:
- A を通常の2倍当たりやすくする
- B を通常の半分にする

「しっかり確率を調整したい」場合にも、  
「なんとなく当たりやすくしたい」場合にも使えるようにしています。

---

### 4. ログ履歴表示

抽選結果はログとして履歴に残せます。

- 何が選ばれたかを確認できる
- 配信中に過去結果を見せられる
- ログの表示位置や見え方も調整可能

ログ表示は、ルーレットとの重なり方や表示方法もある程度調整できます。

---

### 5. 表示方法の調整

文字の表示やルーレットの見え方を変更できます。

主な調整例:

- テキストの表示モード変更
- 文字方向の変更
- ドーナツ穴の有無
- 背景透過
- ポインター位置の調整
- 各種表示サイズの変更

まだ調整中の部分もあるため、今後も少しずつ改修していく予定です。

ルーレット以外を非表示にする「ルーレット以外非表示」モードも用意しています。  
このモード中にどのパネル・要素を表示しておくかを個別に設定できます。

また、一定時間操作がなかった場合にアプリ全体を自動で非表示にする「自動全面非表示」機能もあります。  
非表示までの秒数・フェードアウトの有無などを管理パネルから設定できます。

---

### 6. デザイン編集

色・フォント・セグメントカラーを専用のデザインエディタで編集できます。

- ルーレット・パネル・結果表示の配色変更
- フォント種別の変更
- デザインプリセットの保存 / 切り替え / 複製 / 削除
- セグメントカラーのカスタマイズ

設定パネル内の「デザインエディタを開く」ボタンから起動できます。

---

### 7. 効果音・演出

スピン中や決定時の効果音・演出を変更できます。

#### 効果音
- 内蔵効果音の切り替え
- 消音
- カスタム音声の利用（対応環境時）
- スピン音・決定音・演出音の音量を個別に調整できます

#### 特殊演出
スピン中・停止時に発生する特殊演出（フラッシュ・発光・カットイン・テキストなど）をそれぞれ設定できます。

- 演出ごとの ON / OFF
- 発火確率・タイミングの範囲指定
- バリアント（演出のパターン）の選択

設定パネルの「演出」セクションから調整できます。

---

### 8. 項目の分割

1つの項目を複数のセグメントとして分割表示できます。

例:
- `A` を 3 分割
- `B` を 2 分割

これにより、

- 見た目として分散して配置したい
- 同じ項目を複数箇所に見せたい
- 演出的に散らしたい

といった使い方ができます。

※ 最終結果は、停止時点でポインターが指しているセグメントの元項目で判定されます。

---

### 9. 配置のランダム化

ルーレット内の並び順をランダムに変更できます。

#### 1回だけランダム
現在の並びをその場でランダム化します。

#### 常時ランダム
スピン開始のたびに、配置をランダムに変更します。

「毎回見た目を変えたい」  
「同じ並びで固定される印象を減らしたい」  
といった用途向けです。

---

### 10. マルチルーレット

複数のルーレットを同時に管理・表示できます。

- ルーレットの追加 / 削除
- ルーレットごとの表示 / 非表示切り替え
- 管理パネルでアクティブなルーレットを切り替え
- 各ルーレットで項目リスト・設定・ログを独立して管理
- ルーレット名のダブルクリック編集（管理パネル）
- 書き出し / 読み込みによる移行・共有

管理パネル（F1）の「ルーレット管理」セクションから操作できます。

---

### 11. 被りなし連続抽選

同じ項目が連続して当選しないよう制御しながら、指定回数の抽選を自動で進行できます。

- 実行回数を指定して連続スピン
- 一度当選した項目を次回以降の対象から外す
- 進捗・抽選順の表示
- 途中で中断可能

実行パネル（F5）または右クリックメニューの「被りなし連続抽選...」から起動できます。

---

### 12. チケット管理

チケットパネル（F4）で、効果付きチケットの発行・使用・履歴管理ができます。

- チケットには名前・発行者・効果を設定できます
- 効果の種類：ポインター移動 / 項目非表示 / 重み係数指定 / 固定確率指定 / 追加確率指定
- 使用・削除の履歴を管理（ダブルクリックで復活）
- テンプレートによるチケットの雛形登録
- チケットデータの書き出し / 読み込みに対応

配信中に視聴者がチケットを使って抽選に干渉する演出などに活用できます。

---

### 13. リプレイ

スピンの記録を保存して後から再生できます。

- スピン結果をリプレイとして保存（最大20件）
- リプレイ管理ウィンドウで一覧表示・再生・削除・名称変更
- リプレイのファイルへの書き出し / 読み込みに対応
- 設定パネルの「リプレイ」セクションから操作できます

---

### 14. 画面OCR取り込み

項目パネル上部の「取」ボタンから、画面上の任意範囲を選択してOCRできます。

- ボタンを押すと画面範囲選択モードに入ります
- 読み取ったテキストは項目パネル内の一時表示に出ます
- 認識しづらい場合は「画像調整」タブで明るさ・コントラスト・しきい値などを調整して再認識できます
- 画像調整では表示倍率の変更、ドラッグ範囲選択、ダイアログ右下からのサイズ変更ができます
- OCRは文字の前後関係や行全体も使って推定するため、範囲が狭すぎると認識精度が大きく落ちます。文字の左右・上下に少し余白を含め、単語または行単位で取り込んでください
- 一部の画面を取り込めない場合は、管理パネルのアプリ設定で「OCRキャプチャ」を「Windows GDI」に変更すると改善する場合があります
- 内容がよければ「コピー」して、任意の項目リストへ貼り付けます

ゲーム内に表示された選択肢をルーレット候補にしたい場合など、手入力を減らす用途向けです。

---

### 15. 外部連携

連携パネル（F6）で、外部ツールからのメッセージを受信してスピンやチケット追加を自動実行できます。

- 受信メッセージの自動解析（スピン / チケット追加 / 不明 の判定）
- 自動実行の ON / OFF 切り替え
- 手動での個別実行
- フィルタリング表示・一括適用・一括削除
- キュー管理（最大10件待機）

## キーボードショートカット

| キー | 機能 |
|------|------|
| F1 | 管理パネルの表示 / 非表示 |
| F2 | 項目パネルの表示 / 非表示 |
| F3 | 設定パネルの表示 / 非表示 |
| F4 | チケットパネルの表示 / 非表示 |
| F5 | 実行パネル（被りなし連続抽選）の表示 / 非表示 |
| F6 | 連携パネルの表示 / 非表示 |
| Esc | 全面非表示（全パネル・ルーレットを一時非表示） |

## 設定・保存

各種設定やデータを保存・復元できます。

- **設定のパターン保存**: 設定パネルの「パターン」セクションで、現在のルーレット設定をプリセットとして保存・切り替えできます
- **ルーレットの書き出し / 読み込み**: ルーレット単位でデータをファイルに書き出したり、別の環境から読み込んだりできます（管理パネル）
- **項目リストの保存 / 読み込み**: 項目リストをテキストで保存し、後から読み込めます
- **設定全体のエクスポート / インポート**: アプリ全体の設定をまとめてファイルに書き出し・読み込みできます
- **ログの出力**: 抽選ログをファイルに保存できます

```bash

python main.py
```

```bash
cd RRoulette/pyside6
```
